# Dream Machine
Link to project: 

## Project Overview

Dream Machine is an interactive AI artwork that explores whether a machine can interpret and reconstruct human dreams.

Users enter a dream description which is analysed using a pre-trained machine learning sentiment model from ml5.js. The system then performs a visible reconstruction process before generating an abstract visual output.

The project investigates the gap between human imagination and machine interpretation, questioning whether artificial intelligence can truly understand subjective experiences such as dreams.

---

## Concept

Dreams are often emotional, symbolic and deeply personal. While humans experience dreams through memory, feeling and subconscious associations, machines operate through data and pattern recognition.

Dream Machine asks what happens when a machine attempts to interpret something it can never actually experience.

Rather than recreating dreams realistically, the artwork translates dream descriptions into computational forms. The resulting visualisation is not presented as the user's actual dream, but as a machine reconstruction generated through algorithmic processes.

---

## Machine Learning

The project uses the ml5.js Sentiment Analysis model.

When a user submits a dream, the model analyses the emotional tone of the text and returns a confidence score. This information influences the visual reconstruction generated by the artwork.

The machine learning component is intentionally simple. This supports the critical argument of the project: while the machine appears capable of interpreting dreams, it ultimately reduces complex subjective experiences into computational patterns and probabilities.

---

## Artistic Approach

The visual output is designed to resemble a machine attempting to dream.

Users first encounter a reconstruction phase where the system appears to analyse and process the dream through code-like visual activity. The machine then generates an abstract artwork based on patterns extracted from the text.

The work focuses on interpretation rather than accuracy. The generated output highlights how AI simplifies complex human experiences into patterns, probabilities and visual structures.

---

## Critical Reflection

Dream Machine explores the limitations of artificial intelligence and challenges assumptions about machine creativity.

Although AI systems can generate convincing outputs, they do not possess memories, emotions or lived experiences. The project therefore positions AI as a system of interpretation rather than understanding.

The artwork ultimately asks whether machines can genuinely imagine, or whether they simply transform human input into new configurations of data.

## AI Disclosure

This project uses the pre-trained Sentiment Analysis model provided by ml5.js.

The machine learning model analyses the emotional tone of user-submitted dream descriptions and influences the visual reconstruction generated by the artwork.

No generative image models, large language models, or external AI APIs are used within the final artwork. All visual outputs are generated through custom JavaScript logic and procedural visual systems created by the author.

The project critically examines the limitations of AI interpretation and does not present machine-generated outputs as objective representations of human dreams.
